# 画像フィルタアプリ

選択した画像にフィルタリングをかけることができます  

## Qiita  

画像処理についてQiitaにまとめているので是非ご覧ください。

JavaScript  
Nextjs

## 種類

グレースケール化  
明るさ＋ 50  
明るさ-50  
濃淡の反転  
ポスタリゼーション  
平均化フィルタ  
ラプラシアンフィルタ  
エンボス処理
